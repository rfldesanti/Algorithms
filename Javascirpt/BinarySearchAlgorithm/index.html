The Binary Search Algorithm

This page shows the output of the binary search algorithm written in Javascript

How does this algorithm works? Why is it called "binary search"?

The binary search algorithm works by reducing 50% of the itens (or values) that must be analysed in an ordered list to find the sought value. At each time that the algorithm try to find the item (or the value), 50% of the itens are discarded, making this kind of search much more efficient than a linear search (that analyse individually each value from the list to find the sought value).
